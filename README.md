# Description

A game based on the Monty Hall paradox.

## Technologies

Vue, Vutify, Pinia

## Project setup

```
# yarn
yarn

# npm
npm install
```

### Compiles and hot-reloads for development

```
# yarn
yarn dev

# npm
npm run dev
```

### Compiles and minifies for production

```
# yarn
yarn build

# npm
npm run build
```

### Lints and fixes files

```
# yarn
yarn lint

# npm
npm run lint
```
